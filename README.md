# Galería de Fotos - Ionic

Proyecto desarrollado para la materia Desarrollo de Aplicaciones Móviles Híbridas.

Tecnologías utilizadas:
- Ionic
- Angular
- Capacitor
- Android Studio

Funcionalidades:
- Captura de imágenes desde la cámara
- Almacenamiento en galería interna
- Ejecución en navegador y emulador Android

Para ejecutar el proyecto:

npm install
ionic serve
